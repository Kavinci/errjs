ErrJS

ErrJS is a cross platform error handling solution that provides more information than the basic Javascript Error Class.

Linting is based on Google's Standards. Documentation is based on Google's Standards and compiled and formatted using JSDoc.

src/init.js

Overrides the default On Error Event and creates a more verbose logging solution by initializing the ErrJS module. ErrJS will output to the console for browser uses and either a file or the console when used with NodeJS.
